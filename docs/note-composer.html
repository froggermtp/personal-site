Using Brave?

Brave disables the File System Access API by default, so saving falls back to file downloads. To save notes directly into the repo instead: open brave://flags, search for "File System Access API" (brave://flags/#file-system-access-api), set it to Enabled, and relaunch Brave.

Meanwhile, to stop the save dialog appearing for every downloaded file, turn off "Ask where to save each file" in brave://settings/downloads — downloads then go straight to your Downloads folder.

Two save modes: in browsers that support the File System Access API (Chromium-based, e.g. Chrome/Edge), you can pick this repo's root folder once and notes/photos are written straight into it. In other browsers, Save downloads the note (and photo, if any) as files for you to move into place by hand. On Brave, direct saving needs the "File System Access API" flag enabled in brave://flags first. Either way, nothing here is uploaded anywhere — there is no server and no network request involved.
